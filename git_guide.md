# Git

**Начало работы с GIT:**

1. Настройка конфига
```Bash
git config --local user.name "user";
git config --local user.email "email"
```
2. Клонирование проекта из удаленного репозитория
```Bash
cd folder_name;
git clone https://github.com/Us0kerra/Sports-exercises-Working-title-.git
```
